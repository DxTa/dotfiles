Keep it simple, stupid

Installation:

    git clone git://github.com/tungd/dotfiles ~/cli

    cd ~/cli
    git submodule init
    git submodule update

    source ./install

    vim-bundler
    emacs --batch --quick --load ~/.emacs.d/packages.el

Config included:
- Awesome WM
- Bash
- Cmus
- Emacs
- Irssi
- Tmux
- Vim
- Zsh
